This portfolio encapsulates the core philosophy of my work: **Robust Architecture, Data Integrity, and Actionable Intelligence.**

## 📁 Internship Projects

Below are the links to the four primary project repositories I completed, each building upon the skills from the previous:

### 1. [Code Repository Professionalization (ApexPlanet-Task4)](https://github.com/sricharan-A/ApexPlanet-Task4)
- **Objective:** Establish industry standards for a codebase.
- **Key Contributions:** Refactored project structure, improved error handling and configuration management, implemented CI/CD pipelines, and ensured professional software development practices.

### 2. [Smart City Simulator & Distributed Storage (smartcityHbase)](https://github.com/sricharan-A/smartcityHbase)
- **Objective:** Design data infrastructure for Smart City IoT data.
- **Key Contributions:** Created the simulator framework, integrated HBase for highly scalable distributed storage of sensor data, and documented the architecture.

### 3. [Smart City Data Engineering Pipeline (Smart-City-Repo)](https://github.com/sricharan-A/Smart-City-Repo)
- **Objective:** Professionalize and scale the Smart City Simulator.
- **Key Contributions:** Implemented multi-threading, advanced sensor types (Water Quality, Air Quality), intelligent anomaly simulation, robust logging, and graceful shutdowns to mimic a true production environment.

### 4. [End-to-End ML Pipeline using GCP Cloud Services](https://github.com/sricharan-A/End-to-End-ML-pipeline-using-GCP-cloud-services)
- **Objective:** Predict shipment delivery delays using cloud-based ML.
- **Key Contributions:** Engineered an entire pipeline using GCP (Compute Engine, GCS, Vertex AI Workbench). Uploaded demographic/shipping data, built and trained an optimized XGBoost model, evaluated model performance, and displayed comprehensive results on a VM dashboard.
